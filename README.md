# 🍽️ Foodella — Restaurant Website

Foodella is a fully responsive, front-end restaurant website built with a modern glassmorphism design. It includes a landing page, an interactive menu with cart functionality, and smooth animations — all built using HTML, CSS, and JavaScript.

**Live Demo:** https://foodella.vercel.app/

---

## ✨ Features

### 🏠 Home Page
- Animated hero section with live open/closed status badge
- About Us section with restaurant stats
- Interactive food gallery linked to menu categories
- Auto-playing testimonials carousel (swipe-enabled on mobile)
- Contact form with instant feedback toast
- Newsletter signup section

### 🍔 Menu Page
- Live search bar to filter dishes by name
- Category filter pills (Breakfast, Desi Foods, Fast Foods, Chinese Foods, Desserts & Drinks, Icecreams)
- "Favorites" system — save your favorite dishes (persists via `localStorage`)
- "Chef's Pick" highlighted items
- Order popup with quantity selector and special instructions
- Fully functional shopping cart (add, update quantity, remove, checkout)
- Cart data synced across pages using `localStorage`

### 🎨 Design
- Consistent glassmorphism UI across cards, buttons, and navigation
- Custom color palette tied to the brand's signature red
- Smooth scroll-reveal animations
- Fully responsive across mobile, tablet, and desktop

---

## 🛠️ Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom properties, glassmorphism, animations, responsive layout
- **JavaScript (Vanilla)** — cart logic, filtering/search, carousel, form handling
- **Bootstrap 5** — grid system and base components
- **Bootstrap Icons** — iconography
- **Google Fonts** — Playfair Display & Outfit
